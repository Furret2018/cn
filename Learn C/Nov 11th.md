解决谷歌浏览器chrome置顶显示在最前面问题
chrome有时会总显示在应用最上面，每次切换到别的应用都得关闭或者最小化chrome，很麻烦。
解决方法： Ctrl + Alt + Esc 即可






在使用google chrome浏览器的时候，发现chrome被默认置顶，取消chrome默认的方法为在浏览器上按 “ALT + Space + C”，然后再重开chorme就可以了。
